# 📊 E-Commerce Sales Dashboard (Power BI)

## 🎯 Objective

To analyze e-commerce sales data and generate actionable insights on sales performance, customer behavior, and profit trends using Power BI.

---

## 📂 Dataset Description

The dataset includes:

* Order ID, Order Date
* Product Category & Sub-category
* Sales, Profit, Quantity
* Customer details
* Payment Mode
* Region & State

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning)
* DAX (Measures & Calculations)
* Excel / CSV Dataset

---

## 🔍 Key Analysis Performed

* 📈 Sales & Profit Trend Analysis (Monthly)
* 🗺️ Region-wise & State-wise Sales Analysis
* 🛍️ Category & Sub-category Performance
* 💳 Payment Mode Analysis
* 👥 Customer Contribution Analysis

---

## 📊 Dashboard Features

* KPI Cards (Total Sales, Profit, Quantity, AOV)
* Interactive Filters (Region, Category, Time)
* Bar Charts, Pie Charts, Line Charts
* Drill-down functionality for deeper insights

## 🚀 Key Insights

* 📌 Highest sales observed during festive months (Oct–Dec)
* 📌 Certain product categories contribute major revenue share
* 📌 Few states dominate total sales → expansion opportunity
* 📌 Specific customers contribute high revenue (top customers)

---

## 💡 Business Impact

* Helps businesses identify high-performing products
* Supports decision-making for marketing strategies
* Enables tracking of sales trends and profitability
* Improves customer targeting and retention strategies
