#  Ecommerce Sales Dashboard (Power BI Project)

##  Project Overview

This project focuses on analyzing ecommerce sales data using Power BI to extract meaningful business insights. The dashboard helps in understanding sales performance, profit trends, and customer behavior.

---

## 🎯 Business Problem

The objective of this project is to:

* Monitor overall sales and profit
* Identify top-performing product categories
* Analyze monthly sales trends
* Understand customer purchasing patterns
* Evaluate regional performance

---

## 📊 Key Insights

* 📈 Total sales and profit overview
* 🛍️ Category-wise sales performance
* 📅 Monthly sales trend analysis
* 🌍 Region-wise performance comparison
* 👥 Customer segmentation insights

---

## 🛠️ Tools & Technologies Used

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

## 🔄 Steps Performed

1. Imported dataset into Power BI
2. Cleaned data (removed null values and duplicates)
3. Transformed data using Power Query
4. Created relationships (Data Modeling)
5. Developed calculated columns and measures using DAX
6. Built interactive dashboard with visuals
